How to use

1 · Pick a cell. Inserts always land at the selected cell, and the selection moves below afterwards so repeated inserts stack neatly.

2 · Search. Type words ("world cup 2030") and filter with the chips — open markets only, Yes/No, or multiple choice.

3 · Insert. On each result: % a live probability · Row question, probability, 24h volume, close date, link · Answers the answers × probabilities table (spills downward) · Spark a mini price-history chart.

4 · titles. With the titles toggle on, the market's name is written beside the number (and portfolios get an @username heading), so a stack of inserts stays readable.

5 · Portfolio. Load any Manifold username below, then insert their summary or top positions.

6 · Stay fresh. ↻ re-fetches every MANIFOLD formula now; the auto menu keeps doing it in the background — every 5 minutes up to daily — even with the sheet closed. The Refresh log below records each run (auto or manual) so you can confirm the background refresh is working.
